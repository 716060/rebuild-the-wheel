# libevent 

* 安装

    ```
    tar -zxvf   xxxx
    cd libevent-1.4.14b-stable
    ./configure
    sudo make 
    sudo make install
    ln -s /usr/local/lib/libevent-1.4.so.2 /usr/lib/libevent-1.4.so.2
    ```
